## ⚡ Quick Start (3 Steps)

### 1. Add Your Profile Image

```
Copy your profile photo to:
portfolio-website/public/profile.jpg

Size: 800×800px recommended
Formats: JPG, PNG, or WebP
```

### 2. Install & Run

```bash
cd portfolio-website
npm install
npm run dev
```

Visit: http://localhost:3000

### 3. Deploy

Push to GitHub → Deploy to Vercel → Done!

## 🆘 Troubleshooting Quick Fixes

**Image not showing?**
→ Check file is at: `public/profile.jpg`

**Changes not appearing?**
→ Restart: Save file, then `Ctrl+C` and `npm run dev`

**Port 3000 in use?**
→ Use different port: `npm run dev -- -p 3001`

**Build errors?**
→ Clear cache: `rm -rf .next node_modules && npm install`

See **README.md** for full troubleshooting!
